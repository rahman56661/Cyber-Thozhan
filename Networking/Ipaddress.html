What is an IP Address?

Definition

Internet-la connect aagura ellarum — mobile, laptop, Tab, router, Wifi use panni dhan connect pannuvom appo andha andha device ah unique ah identify panna use pandradhu dhan IP Address. Ithu device-ah identify panna, data enga send panna nu internet-ku sollum.

Example

Namma veedu address irukku la Postman letter anupum bodhu andha address pathu than post ah delivery pannuvan. Adhe maari internet-la message, video, data ellathayum correct device-ku reach panna IP address veedu address maari work aagum.

Real Life Use

What is LAN?

Local Area Network - Connecting Devices in a Limited Area

Definition

A Local Area Network (LAN) is a network used to connect multiple devices in a small geographical area for communication.

Simple explanation: When devices in a home, office, school, or building connect together to share resources and communicate.

What is "Local Area"?

Local area refers to a limited geographical space such as:

  • Home
  • Office
  • School
  • College lab
  • Cafe
  • Single building

Main idea of LAN: Connect devices over short distances for fast data transfer.

LAN Characteristics

Small Geographical Area

Limited area like home, office, or building.

High-Speed Data Transfer

LAN speeds range from 100 Mbps to 10 Gbps, much faster than typical internet speeds.

Multiple Device Connectivity

Various devices can connect to a LAN for communication and resource sharing.

Devices That Connect to LAN

Computers

Laptops

Phones

Printers

CCTV Systems

Smart TVs

Servers

Routers

Switches

LAN Types (2 Main Types)

Wired LAN

Uses Ethernet cables for connection.

Advantages: Stable + Fast connection

Wireless LAN (WLAN)

Uses Wi-Fi for connection.

Advantages: Flexible + Easy to connect

LAN Examples (Real Life)

Home LAN

Router connecting phones, laptops, TVs, and cameras with high speed for local sharing.

Office LAN

20-200 computers, printers, file servers, database servers, and CCTV systems all connected in one LAN.

School/College Computer Lab

50 computers connected to a switch forming a LAN for educational purposes.

LAN Use Cases

File Sharing

Fast file transfer between computers on the same LAN.

Printer Sharing

Multiple employees sharing one printer.

Gaming

LAN gaming (Counter Strike, Valorant private servers) with no lag due to direct LAN speed.

Server Access

Company employees accessing internal websites and shared folders through LAN.

Internet Sharing

Router distributing internet access to all devices on the LAN.

LAN Equipment

What you need to set up a LAN:

  • Router - Distributes internet connection
  • Switch - Increases device connection slots
  • Ethernet Cable - For wired connections
  • Wi-Fi Access Point - For wireless LAN
  • Network Interface Card (NIC) - Present in every device

LAN Advantages & Disadvantages

Advantages

  • Very high speed
  • Secure (limited to local area)
  • Easy maintenance
  • Good sharing options (printer, files, server)
  • Zero data transfer cost

Disadvantages

  • Setup cost (router, switch, cable)
  • Cable management difficult in wired LAN
  • Network maintenance needed in large offices
